Explicação de Módulo

O script principal da página script.js importa um segundo arquivo de funções.

Para isso, o script.js deve ser um módulo, ou seja, deve ser importado com o atributo type="module".

Para importar um módulo, usamos a função import() que retorna uma Promise.

Exemplo de Código


<!-- index.html -->
<script type="module" src="js/script.js"></script>

<!-- script.js -->
import * as func from "./funcoes.js";

const carrosContainer = document.getElementById("carrosContainer");

func.adicionaCarros(carrosContainer);

<!-- funcoes.js -->
export function adicionaCarros(carrosContainer) {
    let carros = ['Gol', 'Voyage', 'HB20', 'Civic'];
    carros.forEach((item) => {
        let p = document.createElement("p");
        p.innerHTML = item;
        carrosContainer.appendChild(p);
    });
}
