History API

Com o History API, conseguir manipular os botões de avançar e voltar de uma página.

Também é possível modificar o link da página em tempo real com o pushstate().

Ok! Mas para oque vou usá-lo?

Juntando este método com a API Fetch, é possível chamar/carregar uma página sem precisar recarregá-la!

Muito massa não é? A vantagem é que o seu site fica muito mais dinâmico, e durante o carregamento você pode adicionar um preloader.

Lembre-se que o tempo de carregamento depende muito do peso da página!

Como utilizar:

                
window.history; //acessa o objeto com várias propriedades
window.history.back(); //acessa a página anterior
window.history.forward() //acessa a próxima página

window.history.pushState(obj, title, url) //permite modificar a url atual do site

//Com o evento popstate é possível executar alguma função ao mudar de página
//Ele irá funcionar apenas se o pushState estiver disponível

window.addEventListener('popstate', mensagem);

function mensagem(){
    console.log('Mudou de página!');
}
