API Fetch - Javascript

A API Fetch do Javascript, é utilizada para realizar requições HTTP a uma API, página ou arquivo de forma simples, retornando uma promisse.

Nos dias de hoje, é mais comum utilizarmos a API Fetch do que o AJAX, devido a sua simplicidade de uso.

Utilizando a API gratuita do VIA CEP, temos um exemplo de código que exibe o logradouro / rua do CEP:17010001

                
let cep = "17010001";
const logradouro = fetch(`https://viacep.com.br/ws/${cep}/json/unicode/`)
                    .then( type => { return type.json(); }  )
                    .then( body => { console.log(body.logradouro)});
//Exibe: Rua Batista de Carvalho

A Promisse possui alguns estados:

  • Pending - É o estado inicial de qualquer promisse, significa que ela ainda não foi resolvida nem rejeitada.
  • Fulfilled - Houve Sucesso na operação, mas não sinifica que vamos ter um retorno. Pois devemos se atentar aos códigos de status de respostas HTTP:
    • 100 - 199 - Informações
    • 200 - 299 - Sucesso
    • 300 - 399 - Redirecionamentos
    • 400 - 499 - Erros client side
    • 500 - 499 - Error server side
  • Rejected - Houve uma falha na operação.
